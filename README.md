# 🚀 EduQuest 20 - O Teu Portal de Desafios

**EduQuest 20** é uma plataforma de jogos educativos desenvolvida especificamente para o público pré-adolescente. Combina conhecimento geral, lógica e ciência com uma interface moderna e mecânicas de gamificação de alto nível.

## 🎮 Funcionalidades Principais
* **20 Mini-Jogos:** Desde Geografia a Programação e História.
* **Sistema de XP e Níveis:** Progressão visível que recompensa o esforço.
* **Economia de Gems:** Ganha moedas para gastar na loja de Power-ups.
* **Mobile-First:** Otimizado para Tablets (iPad/Android) e Smartphones.
* **Privacidade Total:** Todo o progresso é guardado localmente no teu dispositivo.

## 🛠️ Tecnologias Utilizadas
* HTML5 / CSS3 (Glassmorphism UI)
* Vanilla JavaScript (ES6+)
* Web Storage API (Persistência de dados)
* Emoji-based Graphics (Zero dependência de ficheiros externos)

## 📥 Como Jogar
Basta abrir o ficheiro `index.html` em qualquer navegador moderno ou aceder ao link do GitHub Pages.

> **Dica:** No tablet, seleciona "Adicionar ao Ecrã Principal" para jogares em ecrã inteiro como se fosse uma App real!
